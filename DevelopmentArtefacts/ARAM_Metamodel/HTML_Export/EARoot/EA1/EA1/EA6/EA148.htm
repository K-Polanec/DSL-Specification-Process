Attribute Scope Type
isReadOnly Public
Notes: If true, this Activity must not make any changes to objects. The default is false (an Activity may make nonlocal changes). (This is an assertion, not an executable property. It may be used by an execution engine to optimize model execution. If the assertion is violated by the Activity, then the model is ill-formed.) <br/>
isSingleExecution Public
Notes: If true, all invocations of the Activity are handled by the same execution.<br/>
